# 📌 Project Overview

Customer churn is one of the biggest business challenges in telecom and subscription-based industries.

This AI-powered system predicts whether a customer is likely to churn using Machine Learning techniques and an interactive Streamlit web application.

✅ Exploratory Data Analysis

---

# 💡 Business Impact

In customer retention systems, identifying customers who may leave is far more valuable than maximizing overall accuracy.

This project intentionally prioritizes **higher recall** to capture more potential churners, helping businesses take proactive retention actions.

By lowering the prediction threshold from **0.50 → 0.45**, the model achieved:

✅ ~2.6× improvement in churn recall
✅ Better retention targeting
✅ More business-focused predictions
